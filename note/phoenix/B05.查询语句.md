## 1.1 分页查询
使用limit和offset可以快速进行分页。

limit表示每页多少条记录，offset表示从第几条记录开始查起。
``` 
-- 第一页
select * from ORDER_DTL limit 10 offset 0;
-- 第二页
-- offset从10开始
select * from ORDER_DTL limit 10 offset 10;
-- 第三页
select * from ORDER_DTL limit 10 offset 20;
```
