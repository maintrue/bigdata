# 1 Hue集成Impala

## 1.1 修改Hue.ini
```
cd /export/servers/hue-3.9.0-cdh5.14.0/desktop/conf

vi hue.ini

[impala]
server_host=node-3
server_port=21050
impala_conf_dir=/etc/impala/conf
```

## 1.2 重启Hue
```
cd /export/servers/hue-3.9.0-cdh5.14.0/

build/env/bin/supervisor > logs/hue.log 2>&1 &
```
页面访问路径：

http://node-1:8888
